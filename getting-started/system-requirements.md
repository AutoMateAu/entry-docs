# System Requirements

Entry is a cloud-hosted platform — no software installation is required. Access it from any modern web browser.

---

## Browser Support

| Browser           | Minimum Version |
| ----------------- | --------------- |
| Google Chrome      | 90+             |
| Microsoft Edge     | 90+             |
| Mozilla Firefox    | 90+             |
| Safari             | 14+             |

---

## Email Provider

| Provider              | Status      |
| --------------------- | ----------- |
| Microsoft Outlook 365  | Supported   |
| Gmail                  | Planned     |

{% hint style="info" %}
Email integration requires a **Microsoft 365 Business** or **Enterprise** account with Exchange Online. Personal Outlook.com accounts are not supported.
{% endhint %}

## Network

* HTTPS access to `app.askentry.com`
* No VPN or firewall exceptions required for standard use
* Webhook endpoints require inbound HTTPS for Outlook push notifications
