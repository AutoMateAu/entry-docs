# Website Chatbot

Entry provides an AI-powered chat widget you can embed on your agency's website. The chatbot uses your Knowledge Brain to answer visitor questions in real time.

---

## How It Works

```
Visitor Question → Knowledge Brain Search → GPT-4 Response → Chat Widget
```

The chatbot searches your uploaded documents and knowledge summary to generate accurate, agency-specific responses. It handles common enquiries about listings, services, office hours, and policies.

---

## Setting Up the Chatbot

1. Go to **Settings → Chatbot**
2. Copy the embed code snippet
3. Paste it into your website's HTML (typically before the closing `</body>` tag)
4. The chat widget will appear in the bottom-right corner of your site

{% hint style="info" %}
The chatbot widget works on any website — WordPress, Squarespace, custom HTML, or any platform that allows custom scripts.
{% endhint %}

---

## What the Chatbot Can Do

| Capability             | Description                                               |
| ---------------------- | --------------------------------------------------------- |
| **Answer FAQs**         | Responds using your Knowledge Brain documents             |
| **Property enquiries**  | Answers questions about listings if you've uploaded them   |
| **Office information**  | Hours, location, contact details from your docs           |
| **Lead capture**        | Collects visitor contact information                      |

---

## Customisation

The chatbot appearance and behaviour can be configured:

| Setting           | Options                                     |
| ----------------- | ------------------------------------------- |
| **Welcome message** | Custom greeting when the widget opens       |
| **Theme colour**    | Match your agency's brand colours           |
| **Position**        | Bottom-right (default) or bottom-left       |

---

## Improving Chatbot Responses

The chatbot is only as good as your Knowledge Brain content. To improve responses:

* Upload comprehensive FAQs covering common visitor questions
* Add current listing descriptions
* Include office hours, contact details, and location information
* Keep documents up to date — especially pricing and availability

{% hint style="warning" %}
If the chatbot doesn't have relevant information in the Knowledge Brain, it will indicate that it cannot answer the question rather than making up information.
{% endhint %}
